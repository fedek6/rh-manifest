## Recalculation of font-size Photoshop -> CSS
Go to http://www.endmemo.com/sconvert/pixelpoint.php recalculate PX (PS) to Point.

After that go to http://pxtoem.com/ and calculate PT to EM.

### Example:

Formula:
```
1 px =   0.75 point
```
